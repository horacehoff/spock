# Compute
> Very WIP

- Inspired by Rust & Python
- ~5-6x faster than Python
- Does NOT support nested functions
- Terrible error handling

## Syntax examples
```
let x = 20;
// parentheses are optional
if (x == 20) {
print("TRUE!");
}
```
```
let x = 0;
// parentheses are optional
while (x < 10) {
print(x);
x = x+1;
}
```
```
let x = [10, 20, 30, 40];
print(x[0]);
```
